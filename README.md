#  Sistema Cultivar Frotas

Sistema web para gestão de frotas e patrimônio desenvolvido com Flask e SQLite.

##  Funcionalidades

- Cadastro de veículos
- Controle de manutenção
- Relatórios financeiros
- Controle de ferramentas
- Gestão de técnicos
- Histórico de serviços
- Dashboard com gráficos
- Controle de status de pagamento

## Tecnologias

- Python
- Flask
- SQLite
- HTML
- CSS
- Bootstrap
- Chart.js

##  Imagens do Sistema

Sistema desenvolvido para controle interno de frota e patrimônio empresarial.

##  Como executar

```bash
pip install flask
python app.py
```

Acesse:

```text
http://127.0.0.1:5000
```
